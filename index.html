Video Element

<Video Tags>

Tag Name Description
<video> Video Element that allows you to embed a video on your webpage
<p> Paragraph Placed within video tags this is called fallback content. It is displayed if the browser accessing the page doesnt support the video element
<source> Source As a child of the video element it can be used to point to sources for video outside fo the video tag

"Video Attributes"

Attribute Name Description
src Source Contains a path to the video you want to embed in your webpage
controls Controls Allows you to control videoplayback, and adjust volume as well as toggle fullscreen
width Width Controls the width of the video
height Height Controls the height of the video
autoplay Autoplay Makes the video start playing right away, while the rest of the page is loading
loop Loop Makes the video start playing again whenever it finishes
muted Muted Makes the video play with the sound muted by default
preload Preload Used for buffering large files
poster Poster URL of an image displayed before the video is played, a splash/advertising screen
